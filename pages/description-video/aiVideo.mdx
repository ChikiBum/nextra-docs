# AI video 
AI video це функціональність із використанням штучного інтеллекту для генерування відео із текста взятого із статті 


## 1. Додовання сайту
- авторизуватись в адмінку 
- перейти до разділу "_Ai Video_"
- по дефолту відкривається "Sites", якщо інше то перейти до "Sites"
- натиснути кнопку "Create"
- заповнити всі необхідні поля. Обовязкові поля помічени зірочкою `*` . 

_`Owner`_ - це  
_`Site`_ - сюди вказуємо адресу сайту.  
_`Is test Site`_ - це не зрозуміло на що впливає.  
_`Article CSS Selector`_ - тут має бути вказаний CSS selector по якому на сторінці сайту парсер знайде статтю. По дефолту tag `<article>_`.     
_`Selectors to Skip`_ - при необхості вказуємо CSS selector частин встатті яки потрібно пропустити при парсингу. 
_`Text Selector`_ - це CSS selector тексту в статті. По дефолту tag `_<p>_`.  
_`Image Selector`_ - це CSS selector зображень в статті.  
_`Logo preview`_  - сюди завантажуємо логотип.  
_`Source ID`_ - це  
_`Player ID`_ - це  
_`Add InterScenes with Animated Logo`_ - додавати чи ні анімоване лого в відео.   
_`Active Templates`_  - обираємо темплейт для відео.

## 7. Спецільні можливості
### 7.1. Повторний запуск генерації відео 


```html filename="regenerate video link" copy
  https://video-generator-service.bidmatic.io/v1/article/ARTICLEID/process
```

Замість `ARTICLEID{:jsx}` підставити номер статті із довідника `Ai Video` -> `Articles`
