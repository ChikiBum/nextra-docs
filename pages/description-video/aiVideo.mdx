# AI video 
AI video це функціональність із використанням штучного інтеллекту для генерування відео із текста взятого із статті 


## 1. Додовання сайту
- авторизуватись в адмінку 
- перейти до разділу "_Ai Video_"
- по дефолту відкривається "Sites", якщо інше то перейти до "Sites"
- натиснути кнопку "Create"
- заповнити всі необхідні поля. Обовязкові поля помічени зірочкою `*` . 

_Owner_ - це  
_Site_ - сюди вказуємо адресу сайту
_Is test Site:_ - це не зрозуміло на що впливає  
_Article CSS Selector:_ - тут має бути вказаний CSS selector по якому на сторінці сайту парсер знайде статтю. По дефолту <article>. 
_Selectors to Skip:_ - при необхості вказуємо CSS selector частин встатті яки потрібно пропустити при парсингу. 
_Text Selector:_ - це CSS selector тексту в статті. По дефолту <p>.  
_Image Selector:_ - це CSS selector зображень в статті.  
_Logo preview:_  - сюди завантажуємо логотип.  
_Source ID_ - це  
_Player ID_ - це  
_Add InterScenes with Animated Logo:_ - додавати чи ні анімоване лого в відео.   
_Active Templates:_  - обираємо темплейт для відео.
