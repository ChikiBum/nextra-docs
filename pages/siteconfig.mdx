## Quick Guide: Adding Base Configuration to Client Site

In general, when we want to add a new configuration parameter, we need to add a new `property` and its corresponding `value`. 
to do this do next steps:
1. Open the admin panel, navigate to `HBMP Marketplace -> Site Settings -> select the pencil icon next to the website
2. Click on the "Add parameter" ->  add the `property` ->  add the `value`

Itâ€™s important to note that the `property` is always a string, but the `value` can be an array, object, string, etc.
