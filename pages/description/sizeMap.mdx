# Size mapping

**sizeMap** - Size mapping grid for different screen sizes in the format of Google size mapping.

Example:

```js filename="sizeMap example" copy
<script>
  "sizeMap": [ 
    [[700],null],
    [[468,510],[[320,165],[300,165],[468,60],[320,100],[320,50],[300,100],[300,50],"fluid"]],
    [[468],[[468,60],[320,100],[320,50],[300,100],[300,50],"fluid"]],
    [[300,510],[[320,165],[300,165],[320,100],[320,50],[300,100],[300,50],"fluid"]],
    [[300],[[320,100],[320,50],[300,100],[300,50],"fluid"]],
    [[0],[[300,100],[320,100],"fluid"]] 
  ]
</script>
```

In this configuration, we have (describing from bottom to top):

- For screen width less than 300px, size: `[320,100]`
- Screen width from 300px to 468px and height less than 510px, sizes: `[320,100],[320,50],[300,100],[300,50]`
- Screen width from 300px to 468px and height greater than 510px, sizes: `[320,165],[300,165],[320,100],[320,50],[300,100],[300,50],"fluid"`
- Screen width from 468px to 700px and height less than 510px, sizes: `[468,60],[320,100],[320,50],[300,100],[300,50],"fluid"`
- Screen width from 468px to 700px and height greater than 510px, sizes: `[320,165],[300,165],[468,60],[320,100],[320,50],[300,100],[300,50],"fluid"`
- Screen width greater than 700px: nothing is displayed

It's important to adhere to the correct format (as shown in the example) when defining size mappings.

## 1. Delault size mapping

### 1.1. Desktop

```js filename="sizeMap example"
<script>
  "sizeMap": [ 
    [[1440, 0], [[1440, 180], [1366, 90], [1280, 90], [970, 90], [980, 120], [1350, 180], [1180, 120]]], 
    [[1366, 0], [[1366, 90], [1280, 90], [970, 90], [980, 120], [1350, 180], [1180, 120]]], 
    [[1280, 0], [[1280, 90],   [970, 90], [980, 120], [728, 90], [1180, 120]]], [[1000, 0], [[970, 90], [980,120], [728, 90]]], 
    [[700, 0], [[728, 90]]] 
  ]
</script>
```
